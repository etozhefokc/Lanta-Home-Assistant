# Lanta Home

Интеграция для управления домофоном Lanta (API dm.lanta-io.ru) в Home Assistant.

## Возможности

- **Выключатели:** настройки домофона — распознавание лиц, VoIP, трубка (CMS), журнал событий, гостевой режим, White Rabbit
- **Сенсоры:** баланс, бонусы, код домофона, ссылка на личный кабинет
- **Кнопки:** открыть дверь, сбросить код
- **Камеры:** просмотр камер с автопродлением токенов

## Установка

Положить папку `lanta_home` в `custom_components/`.

## Настройка

**Настройки → Устройства и службы → Добавить интеграцию → Lanta Home**.

Ввести номер телефона от приложения Lanta и дождаться звонка на 8-800-222-06-68 для авторизации.
