# Lanta Home

Интеграция для управления домофоном Lanta в Home Assistant.

## Возможности

- **Выключатели:** распознавание лиц, VoIP, трубка (CMS), журнал событий, гостевой режим, White Rabbit
- **Сенсоры:** баланс, бонусы, код домофона
- **Кнопки:** открыть дверь, сбросить код домофона
- **Камеры:** просмотр камер

## Установка через HACS

1. Добавьте custom repository: `https://github.com/etozhefokc/Lanta-Home-Assistant`
2. Категория: **Integration**
3. Установите **Lanta Home**
4. Перезапустите Home Assistant

## Настройка

**Настройки → Устройства и службы → Добавить интеграцию → Lanta Home**.

Введите номер телефона от приложения Lanta и дождитесь звонка на **8-800-222-06-68** для авторизации.
